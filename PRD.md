## 1. 产品概述

### 1.1 产品定位

**一句话描述**：聚焦AI领域的垂直招聘岗位聚合平台，帮助求职者一站式获取全网AI岗位信息，降低求职信息获取成本。

### 1.2 核心价值主张

- **信息聚合**：整合全网AI相关岗位信息
- **垂直聚焦**：专注于AI领域，提供精准的岗位筛选和推荐
- **岗位全面**：覆盖AI技术岗与非技术岗，满足各类求职者需求
- **信息降噪**：过滤非AI相关岗位，减少无效浏览时间
- **效率提升**：通过智能匹配和筛选工具，加快求职效率

### 1.3 解决的核心痛点

| 痛点 | 解决方案 |
|------|----------|
| 求职信息分散，需要浏览多个平台 | 聚合全网AI岗位信息，统一展示 |
| 大量非AI岗位干扰检索效率 | AI相关性过滤，精准定位 |
| 非技术背景想进入AI行业却找不到对应岗位 | 新增AI非技术岗分类 |
| 岗位信息质量参差不齐 | 结构化展示，关键信息一目了然 |
| 错过优质岗位更新 | 订阅/推送功能，及时获取新机会 |

## 3. 功能需求

### 3.1 核心功能

#### 3.1.1 岗位聚合与展示

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 多源岗位聚合 | 抓取Boss直聘、猎聘、拉勾等平台AI岗位 | P0 |
| 岗位列表页 | 展示岗位基本信息列表 | P0 |
| 岗位详情页 | 展示完整岗位信息（JD、要求、福利） | P0 |
| 筛选器 | 城市、薪资、经验、学历、公司规模 | P0 |
| 关键词搜索 | 岗位名、公司名、技能关键词 | P0 |
| 岗位收藏 | 收藏感兴趣岗位 | P1 |
| 岗位分享 | 分享到社交平台 | P2 |

#### 3.1.2 智能推荐

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 求职画像 | 填写期望岗位、经验、技能 | P1 |
| 智能推荐 | 基于画像推荐匹配岗位 | P1 |
| 薪资分析 | 对比同岗位市场薪资 | P2 |
| 技能匹配度 | 显示岗位与技能匹配程度 | P2 |

#### 3.1.3 订阅与通知

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 岗位订阅 | 设置条件订阅符合条件的岗位 | P1 |
| 邮件通知 | 每日/每周发送订阅汇总 | P1 |
| 新岗位推送 | 符合条件的即时推送 | P2 |

#### 3.1.4 用户中心

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 注册/登录 | 邮箱、手机号、第三方登录 | P0 |
| 简历管理 | 填写期望岗位、经历、技能 | P1 |
| 收藏记录 | 查看已收藏岗位 | P1 |
| 订阅管理 | 管理订阅条件 | P1 |

### 3.2 岗位分类体系

#### 技术岗

| 分类 | 细分岗位 |
|------|----------|
| 算法工程师 | 机器学习、深度学习、NLP、CV、推荐系统、搜索算法 |

| 自动驾驶 | 感知、规划控制、算法工程师 |

#### 非技术岗

| 分类 | 细分岗位 |
|------|----------|
| 产品经理 | AI产品经理、AIGC产品经理、产品运营 |
| 市场营销 | 品牌营销、数字营销、市场推广、内容营销 |
| 销售 | 售前工程师、解决方案专家、B2B销售 |
| 运营 | 用户运营、内容运营、社区运营、数据运营 |
| 项目管理 | AI项目经理、项目交付、技术项目经理 |
| 其他 | 客户成功、培训师、伦理合规 |

## 6. 竞品分析

| 竞品 | 优势 | 劣势 | 差异化 |
|------|------|------|--------|
| 拉勾网 | 垂直互联网，用户精准 | 非AI垂直 | AI垂直+非技术岗 |
